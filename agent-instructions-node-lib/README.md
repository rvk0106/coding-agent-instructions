# Agent Instructions — Node.js Library

Tool-agnostic instruction system for developing Node.js libraries/packages.

## Installation
```bash
curl -fsSL https://raw.githubusercontent.com/rvk0106/agent-instructions-node-lib/main/quick-install.sh | bash
```

## Quick start
1) Plan: `plan library for TICKET-ID` → `docs/TICKET-ID-plan.md`
2) Execute: `execute plan 1 for TICKET-ID`
3) Verify: `npm test && npm run lint && npm run build`
