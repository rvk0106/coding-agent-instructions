# Agent Instructions — React

Tool-agnostic instruction system for using coding agents safely with React applications.

## Installation
```bash
curl -fsSL https://raw.githubusercontent.com/rvk0106/agent-instructions-react/main/quick-install.sh | bash
```

## Quick start
1) Plan: `plan frontend for TICKET-ID` → `docs/TICKET-ID-plan.md`
2) Execute: `execute plan 1 for TICKET-ID`
3) Verify: `npm test && npm run lint && npm run build`
